\section{零五}
\vspace*{2\ccwd}
\begin{adjustwidth}{2cm}{1cm}
    \Large\kaishu{
    2019年，11月某日

    \vspace*{2\ccwd} 

    勿见你是天神

    念生膜拜

    无敢昵近

    星月遥遥

    乞怜你夜的光耀

    倘遭阴云

    似也为弃——

    这点滴的照抚

    直堕入幽暗无尽

    孤独的忍受

    长长久久

    黎明前的漫漫无眠

    东方的朝晖又起

    已不是你的光

    故，我将在黑昼沉睡

    等来那属于你的光临

    别是你神迹的天降

    \vspace*{2\ccwd} 

    你应如凡，落凡的神

    我的爱更胜从前

    非关信仰

    纯然的爱，只为付予

    如细流汇积

    只因海的唤取

    你自有广博的空间，自由

    我已无踪

    已是你的涓滴血脉

    风狂浪啸

    腾空的机缘

    借以观览

    你无边的美

    碎身亦无憾

    我的一生哪

    在追随你的事业中完满
}
\end{adjustwidth}
