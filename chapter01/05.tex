\section{零五}
\vspace*{2\ccwd}
\begin{adjustwidth}{2cm}{1cm}
    \Large\kaishu{
    某年，9月2日，周二

    \vspace*{2\ccwd} 

    只是不想忘怀

    可曾还有思念

    在这里倍觉孤单

    想起你不由自主

    把过去打包

    寄往未名的地址

    最后会被退回

    可能遗落途中

    不用记得

    曾失去过你

    不曾，记得过你

    \vspace*{2\ccwd} 

    大雨，我漫步

    路水溅湿裤角

    它也没过心角

    湿凉——不是这天气

    是我失落不安的心

    遇见一只小猫

    她也有着不安

    我却不能安抚

    只是路人的我

    于你——

    编号甲或丁

    \vspace*{2\ccwd} 

    我再一次漫步

    漫无目的

    走失，找不回原来的我

    新的开始

    只不过不得已的
    
    结束，结束在那花盛的季节

    没等来结果
}
\end{adjustwidth}
